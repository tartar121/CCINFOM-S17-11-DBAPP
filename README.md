# Pharmacy Management System
This repository contains the database application project in partial fulfillment for the **CCINFOM (Information Management)** course at **De La Salle University.**
The system is built using Java and SQL to provide a structured solution for managing a pharmacy's complex data.

## Key Features
#### Records Management
Create, read, and update records for Medicines, Customers, and Suppliers.

#### Customer Purchases
Process medicine sales, verify customer status (including senior/PWD discounts), and update stock levels accordingly.

#### Supplier Deliveries
Manage the medicine restocking process by recording delivery requests from active suppliers and updating inventory upon completion.

#### Supplier Returns
Handle the return of expired or discontinued medicines to the correct supplier, including verification and stock updates.

#### Reporting
Generate reports for customer purchases, medicine returns, and supplier procurement.
